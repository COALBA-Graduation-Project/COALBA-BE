> 졸업프로젝트 2022.09 ~ 2023.02
>
> COALBA에서는 사장님과 알바생이 함께하는 공간 워크스페이스에서 공유된 스케줄표를 확인할 수 있으며 보다 편리한 대타요청으로 
요청 승인 시 자동으로 스케줄 근무자가 변경됩니다. 또한 비콘 기반 근태관리와 이에 따른 급여확인까지 오직 한 앱에서 이용 가능합니다.
